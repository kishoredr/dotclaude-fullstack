---
name: seo-optimization
description: Use when building pages, reviewing metadata, or auditing site
  structure for search engine and AI search (GEO/AEO) visibility.
---
Every page requires:
- <title>: 50–60 chars, primary keyword first
- meta description: 120–160 chars, action-oriented
- og:title, og:description, og:image (1200×630px)
- twitter:card, twitter:title, twitter:description
- Canonical URL (prevents duplicate content penalties)

Technical SEO:
- sitemap.xml: auto-generated, submit to Google Search Console
- robots.txt: block /api/, /admin/, /_next/
- Structured data: Organization, BreadcrumbList, Article/Product
- Core Web Vitals passing (LCP < 2.5s, CLS < 0.1, INP < 200ms)

GEO / AEO (AI search readiness):
- FAQ sections with explicit Q&A structure
- Entity markup for key concepts
- Clear authorship and publication dates on content
- Cite sources for factual claims
