File locations:
  /docs/API_SPEC.md          API reference (OpenAPI format)
  /docs/ARCHITECTURE.md      Decisions + rationale
  /docs/RUNBOOK.md           Operational procedures
  /docs/DATABASE_SCHEMA.md   Schema + index strategy
  /docs/adr/YYYYMMDD-title.md  Architecture Decision Records

ADR template:
  # ADR-NNN: [Title]
  Date: YYYY-MM-DD  |  Status: Accepted
  ## Context
  ## Decision
  ## Rationale
  ## Consequences

Write for the engineer joining the team next month.
Assume they know the tech stack — not this specific project.
